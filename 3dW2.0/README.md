# Inventario 3D

Sistema de inventario con visualización 3D dinámica. Los estantes se generan automáticamente a medida que agregás productos.

## Características

- 📦 Estantes 3D dinámicos que crecen con tu inventario
- 🖼️ Subí el logo de cada producto y aparecé en la caja 3D
- 📅 Control de fechas de vencimiento con indicadores visuales
- 🎮 Controles de cámara (rotar, zoom, pan)
- 🚀 Hosteado en GitHub Pages
